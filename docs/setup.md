# Setup

This repo is the public, generic OpenClaw chess-coach layer.

## What you need
- Python 3
- OpenClaw workspace capable of running skill scripts
- Optional: a private companion repo with full corpus data for full-scale operation

## What works from this repo alone
- reading the docs
- inspecting the public skills and reusable tooling
- experimenting with the minimal example data in `examples/`

## What requires the private companion repo
- full corpus search across the working `games.pgn`
- full source-list-driven sync workflows
- generated PDF/output review against the real operating data layer
