# Search Examples

This directory preserves reusable query examples, prompt examples, and JSON fixtures that were carried forward from an earlier staging repo.

They are public-safe supporting assets for the `chess_tools/` search stack.

Contents include:
- structured query JSON examples
- fuzzy-query and natural-language prompt examples
- backup-workflow notes
- motif / relation / attack-shape example prompts

These files are examples and reference material.
They are not a private corpus.
